📱 安装后使用

# 查看帮助
server-optimize --help

# 单项优化
server-optimize tcp    # TCP网络优化
server-optimize dns    # DNS服务器配置
server-optimize bbr    # 启用BBR算法
server-optimize ssh    # SSH安全配置

# 全部优化
server-optimize all

# 快捷命令
optimize-tcp          # 直接运行TCP优化
optimize-dns          # 直接运行DNS配置
⚠️ 系统要求： Debian 9+ / Ubuntu 16.04+ / CentOS 7+ | Root权限 | 基本网络连接

新服务器初始化：

# 史上最简单的服务器优化命令

批量服务器部署：

# 批量优化多台服务器
for server in server{1..10}.com; do

done
